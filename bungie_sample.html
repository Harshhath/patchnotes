Destiny 2 Update 9.5.0.5

Jan 13, 2026 - Destiny 2 Team

Activities

Renegades

  • Fixed an issue where the mission Welcome to the Frontier could sometimes soft-lock when returning to the hangar under poor networking conditions.
  • Fixed an issue where claiming engrams from Syndicate Quartermasters generated an empty engram in the "wrapped items" section of the Inventory screen.
  • Fixed an issue where claiming rank up reward engrams from Syndicate Quartermasters required The Witch Queen ownership.
  • Fixed an issue where players logging into multiple characters of the same class in one week could progress or lose Syndicate Weekly Shipment progress.
  • The Salaried Mercenary Triumph is now tracked per account, not per class.
    • Players must log in once on each class for previously earned progress to be counted.
  • Reduced the number of Guardian multikills required for the Knock ‘Em Down Triumph to five (previously 15).

Lawless Frontier

  • Fixed an issue where certain Smuggle encounters could be blocked from completion if relic items were accidentally dropped in inaccessible locations.
  • Fixed a rare issue where Smuggle runs could soft-lock at the start of exfil.
  • Fixed an issue in Sabotage activities where sniping or killing the initial antenna holder extremely fast would cause the encounter to not progress.
  • Fixed an issue in the Venus Grotto Doublecross arena where some Invaders could hide from the other team at the top of the PvP dome. They can't hide anymore.
  • Fixed a rare bug where the Invasion teleporter remained open and allowed Invaders to reuse after returning to the ready room to collect rewards.
  • Fixed an issue where non-invader Walkers would sometimes not have correct damage reduction.
  • Fixed an issue where combatants could teleport inside Sabotage objectives with shields and become unreachable.
  • Fixed a visual issue where Forging Matrix Europa: Sabotage would display the description for a Bounty Hunt activity.

Raids and Dungeons

Equilibrium

  • Reduced the cooldown and increased damage for shadow power to improve its utility.
  • Fixed an issue where the shadow power tutorial would continually resurface on alternate characters.
  • Fixed an issue where the Harrow Apprentice would not pursue players and not progress the encounter.
  • Fixed an issue where the Harrow Apprentice’s shields could not be removed to begin a second DPS phase.
  • Fixed an issue where the shadow power pools visual state was incorrect.
  • Fixed an issue in the final encounter where some pools in the shadow realm were granting less shadow power than intended when gathered quickly after teleporting in.
  • Fixed an issue allowing sunspots and the Kill Shot mod to break Taken Captain immunity shields intended to require shadow power.
  • Fixed an issue where the third puzzle would not reset correctly after a failed completion attempt.

The Desert Perpetual

  • Fixed an issue where the laser attack from Koregos was not activating as intended.
  • Fixed an issue where players would not be granted the Death-Defiant II Triumph upon completing The Desert Perpetual Epic Raid without any fireteam member dying or returning to orbit.

Portal

  • Increased the activity timer in Master Conquest: Derealize.
  • Increased the starting number of revive tokens available in higher-difficulty Arena Ops and higher-difficulty matchmade/playlist Fireteam Ops activities.
  • Fixed an issue in the Heliostat Pinnacle Op where the name of the Psionic Protection buff was not displaying.
  • Fixed an issue where Grandmaster Vanguard Alerts were not available in Fireteam Finder.

Crucible

  • Fixed an issue where Crucible Ops match completions did not contribute to the B Greater weekly objective.
  • Fixed an issue where Trials of Osiris Lighthouse chest would not award shaders as intended.
  • Fixed an issue that allowed players with an active PvP restriction to still access Trials of Osiris.
  • Fixed an issue where lobby balancing was enabled for Trials of Osiris.
  • Fixed an issue where the latest Trials of Osiris Ghost shell was unobtainable for some players.

Gameplay and Investment

Exotic Armor

Titan

Armamentarium

  • Fixed an issue where grenade throws with this Exotic equipped were not reloading Heat frame weapons.

Warlock

Mataiodoxia

  • Fixed an issue where the suspending burst from Stylostixis could chain indefinitely.
    • The weave settles once more...

Weapons

Exotic Weapons

Service of Luzaku

  • Fixed an issue causing threadling nests to lose health more quickly in some activities. Threadling nest lifetime now should not change from activity to activity.

Wish-Keeper

  • Fixed an issue where ensnaring lines were not generating when hitting a target directly.

Praxic Blade

  • Fixed an issue where players could activate the Hyperblade perk without guarding as intended.

Weapon perks

  • Fixed an issue where the Accelerated Assault origin trait would lose ammo on burst weapons.
  • Fixed an issue where the Gutshot Straight perk was doing more damage than intended to Divinity cages.

Quests and Pursuits

  • Fixed an issue where the Solo Ops step of the Exalting the Flame, Taming the Storm, and Into the Abyss New Light quests could not be completed on difficulties higher than Advanced.

General

  • Fixed an issue with the Digi-Teal shader that resulted in a different shade of cyan being applied when compared to the rest of the Aquamarine Assistant pack's contents.
  • Fixed an issue in the Eververse Archive where the Oracular weapon ornament for Conditional Finality incorrectly displayed that it required players to have reached the Lighthouse.
  • Fixed an issue where the final cinematic in The Final Shape campaign lost some visual effects.
